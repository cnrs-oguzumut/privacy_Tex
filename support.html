Frequently Asked Questions

How do I format my LaTeX document?
Open your LaTeX (.tex) file using the "Open LaTeX File" button. The app will automatically format it according to academic standards. You can then save the formatted version using the "Save" button.
How do I format my BibTeX bibliography?
Open your BibTeX (.bib) file using the "Open BibTeX File" button. The app will sort and format your bibliography entries. Save the formatted file when ready.
What file formats are supported?
The app supports .tex (LaTeX) and .bib (BibTeX) files. All processing is done locally on your device for maximum privacy and security.
Is my data safe?
Yes! All file processing happens locally on your Mac. We don't collect, store, or transmit any of your data. See our Privacy Policy for details.
The app won't open my file. What should I do?
Make sure your file is a valid .tex or .bib file. If the problem persists, please report an issue on our GitHub page with details about the file type and any error messages you see.
How do I report a bug or request a feature?
Visit our GitHub Issues page to report bugs or request new features. Please include as much detail as possible.
